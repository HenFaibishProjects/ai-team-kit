folder_open How to View All Projects

Learn how to browse, search, and manage all projects in your organization

rocket_launch Quick Start

The "View All Projects" page gives you a comprehensive overview of all projects across your organization. You can search, filter, and access detailed information about each project.

folder_open Go to All Projects person My Projects Only
search Search & Filter

Search by name: Use the search bar to quickly find projects by name or description.

Filter by status: View projects by their current status (Planning, Active, On Hold, Completed).

Sort options: Sort projects by name, date created, or last updated.

 info Project Information

Project cards: Each project is displayed as a card showing key information.

Status indicators: Color-coded badges show project status at a glance.

Quick stats: See team size, sprint count, and progress indicators.

 touch_app Quick Actions

View details: Click on any project card to see full project details.

Edit project: Access project settings and configuration.

Manage team: Add or remove team members directly from the project card.
list Step-by-Step Guide looks_one Accessing the Projects Page

From the navigation menu:

  1. Click on "Projects" in the main navigation menu
  2. Or click "View All Projects" from the dashboard
  3. The page will load showing all projects in your organization
lightbulb

Tip: You can also access your personal projects by clicking "My Projects" which shows only projects you're a member of.

 looks_two Searching for Projects

Using the search functionality:

  1. Locate the search bar at the top of the projects page
  2. Type the project name or keywords from the description
  3. Results will filter automatically as you type
  4. Clear the search to see all projects again
code

Example: Type "mobile" to find all mobile app projects, or "2024" to find projects created this year.

 looks_3 Filtering by Status

Filter projects by their current status:

  • Planning - Projects in the planning phase
  • Active - Currently active projects
  • On Hold - Temporarily paused projects
  • Completed - Finished projects

Click on any status filter to show only projects with that status.

 looks_4 Viewing Project Details

Access detailed project information:

  1. Click on any project card to open the project details page
  2. View comprehensive information including:
    • Project description and goals
    • Team members and their roles
    • Sprint information and progress
    • Tasks and assignments
    • Project timeline and milestones
  3. Use the tabs to navigate between different sections
  4. Click the back button to return to the projects list
 looks_5 Managing Projects

Project management actions:

  • Edit Project: Click the edit icon to modify project details
  • Add Team Members: Use the team management button to add or remove members
  • Update Status: Change project status as it progresses
  • Archive Project: Move completed projects to archive
warning

Note: Some actions may require admin or project owner permissions.

 help Common Questions What's the difference between "All Projects" and "My Projects"?

"All Projects" shows every project in your organization, while "My Projects" only displays projects where you are a team member.

 Can I see archived projects?

Yes, use the status filter and select "Completed" or look for an "Archived" filter option to view archived projects.

 How do I export the project list?

Look for an export button (usually in the top right) to download the project list as CSV or Excel format.

 Why can't I see some projects?

You may not have permission to view certain projects. Contact your organization admin if you need access to specific projects.
